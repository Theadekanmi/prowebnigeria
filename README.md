# Website Designer Nigeria - Professional Web Design Company

A modern, professional website for Nigeria's leading web design company, built with Next.js 14, Tailwind CSS, and Framer Motion.

## 🚀 Features

### **Core Website Features**
- **Modern Design**: Clean, professional design with smooth animations
- **Responsive Layout**: Mobile-first design that works on all devices
- **Interactive Components**: Engaging user experience with Framer Motion
- **SEO Optimized**: Comprehensive SEO setup for both local (Nigeria) and global search
- **Fast Performance**: Optimized for Core Web Vitals and user experience

### **Sections**
1. **Header**: Sticky navigation with contact info and CTA buttons
2. **Hero**: Animated hero section with rotating text and statistics
3. **Services**: Comprehensive service offerings with pricing
4. **About Us**: Company story, team, values, and achievements
5. **Portfolio**: Filterable project showcase with case studies
6. **Blog**: Content marketing section with articles and insights
7. **Testimonials**: Client reviews and success stories
8. **Contact**: Multiple contact forms and office information
9. **Footer**: Site map, social links, and newsletter signup

### **Technical Features**
- **Next.js 14**: Latest React framework with App Router
- **Tailwind CSS**: Utility-first CSS framework
- **Framer Motion**: Smooth animations and transitions
- **Lucide React**: Beautiful icon library
- **Responsive Design**: Mobile-first approach
- **SEO Optimization**: Meta tags, structured data, sitemap
- **Performance**: Optimized images, fonts, and loading

## 🛠️ Technology Stack

- **Frontend**: Next.js 14, React 18, JavaScript
- **Styling**: Tailwind CSS, CSS3
- **Animations**: Framer Motion
- **Icons**: Lucide React
- **Deployment**: Vercel (recommended), Netlify, or any hosting platform
- **SEO**: Next.js built-in SEO, structured data, sitemap

## 📋 Prerequisites

Before running this project, make sure you have:

- **Node.js** 18.17 or later
- **npm** or **yarn** package manager
- **Git** for version control

## 🚀 Getting Started

### 1. Clone the Repository

```bash
git clone https://github.com/yourusername/websitedesignernigeria.git
cd websitedesignernigeria
```

### 2. Install Dependencies

```bash
npm install
# or
yarn install
```

### 3. Run the Development Server

```bash
npm run dev
# or
yarn dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser to see the result.

### 4. Build for Production

```bash
npm run build
# or
yarn build
```

### 5. Start Production Server

```bash
npm start
# or
yarn start
```

## 🎨 Customization

### **Colors and Branding**
Update the color scheme in `tailwind.config.js`:

```javascript
module.exports = {
  theme: {
    extend: {
      colors: {
        primary: {
          50: '#eff6ff',
          500: '#3b82f6',
          600: '#2563eb',
          700: '#1d4ed8',
        },
        // Add your brand colors
      },
    },
  },
}
```

### **Content Updates**
- **Company Information**: Update contact details, addresses, and company info in components
- **Services**: Modify service offerings and pricing in `Services.js`
- **Portfolio**: Add your projects in `Portfolio.js`
- **Team**: Update team members in `About.js`
- **Blog**: Add your blog posts in `Blog.js`

### **SEO Configuration**
Update SEO settings in `app/layout.js`:

```javascript
export const metadata = {
  title: 'Your Company Name - Custom Title',
  description: 'Your custom description',
  keywords: ['your', 'custom', 'keywords'],
  // ... other metadata
}
```
